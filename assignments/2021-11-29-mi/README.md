### 介紹
練習使用 Express 框架，以餐廳清單系統，可查詢餐廳名稱、餐廳類型、查看詳細資訊。

### 功能
* 查看所有餐廳
* 查看餐廳詳細資訊
* 搜尋特定餐廳名稱
* 搜尋特定餐廳類型

### 使用方式
1. 確認有安裝 node.js 與 npm
2. 將專案clone到本地
3. 在本地開啟後，透過終端機進入資料夾 a1rstaurant
4. 瀏覽器進入網址 http://localhost:3000

### 開發工具
* Node.js 16.13.0
* Express 4.17.1
* Express-Handlebars 3.0.0
* Bootstrap 4.3.1
* Font-awesome 5.15.4
