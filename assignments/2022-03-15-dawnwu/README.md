# 介紹
試作簡易的餐廳清單

# 功能
瀏覽餐廳、餐廳詳細資料、搜尋餐廳、連結地圖。

# 開始使用
1. 先確認有安裝 node.js 與 npm
2. 將專案下載至本地電腦
3. 透過終端機進入資料夾，輸入：
`npm install`
4. 安裝完成後在終端機輸入：
`node app.js`
5. 看到以下訊息表示成功開啟：
`This is localhost:3000`
6. 打開瀏覽器輸入以下網址：
`http://localhost:3000`
7. 關閉伺服器連線請在終端機輸入：
`ctrl+c`
