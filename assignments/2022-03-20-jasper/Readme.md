## 開始使用
---
1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入： `npm install`
4. 安裝完畢後，繼續輸入： `npm run dev`
5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址
`Experss is listening on http://localhost:3000`
6. 若欲暫停使用 `ctrl + c`
---
