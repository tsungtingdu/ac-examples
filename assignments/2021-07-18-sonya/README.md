# 短網址產生器
簡易短網址產生器

## 功能
使用者可以：

- 輸入網址後，系統產生一組短網址
- 輸入的網址可以選用http或https為開頭
- 可以一件複製短網址

## 安裝方式
複製檔案
```
git clone https://github.com/konnono/url-shortener.git
```

在檔案夾中執行
```
npm install
```

執行程式:
```
npm run start
```

以開發模式執行程式(nodemon):
```
npm run dev
```
